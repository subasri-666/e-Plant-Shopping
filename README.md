# e-Plant Shopping

## Overview

e-Plant Shopping is an online plant store built using React. Users can browse various indoor and outdoor plants, add them to the cart, manage quantities, and view the total price.

## Features

* Browse plant collections
* Add plants to cart
* Remove plants from cart
* Update plant quantity
* View total items and total cost
* Responsive user interface

## Technologies Used

* React.js
* Redux Toolkit
* CSS3
* JavaScript (ES6)

## Installation

1. Clone the repository

git clone https://github.com/yourusername/e-plant-shopping.git

2. Install dependencies

npm install

3. Start the development server

npm run dev
